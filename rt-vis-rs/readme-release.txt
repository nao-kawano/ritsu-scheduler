Ritsu Visualizer
==================================================

Ritsu Visualizer is an interactive visualization and schedule design tool for Ritsu Scheduler.

■ How to Run
Double-click "rt-vis.exe" to launch the application.

■ Features
- Create, edit, load, and save schedule configuration files (config.toml).
- Simulate and visualize timelines and load metrics for the configured schedules.

■ Documentation
For more information and updates, please check the GitHub repository:
